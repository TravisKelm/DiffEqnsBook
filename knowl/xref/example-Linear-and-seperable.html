Example 2.4.12.

Solve the initial value problem
\begin{equation} \dfrac{dy}{dt}=\dfrac{2y+1}{t},\ y(1)=0.\tag{2.4.5} \end{equation}
Note this is equation is both linear and separable (we solved it in the previous section). The equation (2.4.5) can be separated as follows:
\begin{equation*} y'-\dfrac{2}{t}\;y=\dfrac{1}{t} \end{equation*}
However, since the equation was initially presented in the standard linear form, in this example we will solve the original equation (2.4.5) using an integrating factor.
🔗
Solution.
  • First, find the general solution.
    🔗
    • Since the equation is in the proper form, we first find the integrating factor:
      🔗
      \begin{equation*} p(t) = -\frac{2}{t}\text{ so }\mu(t) = e^{-\int \frac{2}{t} \, dt}= e^{-2\ln |t|}= t^{-2}. \end{equation*}
      🔗
      🔗
    • Multiplying both sides of the DE by \(\mu(t)\) gives
      \begin{equation*} t^{-2}y'-2t^{-3}y=t^{-3}, \end{equation*}
      🔗
      which we recognize as
      \begin{equation*} \left[t^{-2}y\right]'=t^{-3}. \end{equation*}
      🔗
      🔗
    • Integrate both sides with respect to \(t\) to obtain:
      \begin{align*} \int\left[t^{-2}y\right]'\,dt\amp = \int t^{-3}\,dt \\ t^{-2}y\amp = -\dfrac{1}{2}t^{-2}+c \quad (c\in \mathbb{R}). \end{align*}
      🔗
      🔗
    • Solving for \(y\text{,}\) we find the general solution of the DE:
      \begin{equation*} y=ct^{2}-\dfrac{1}{2}\quad (c\in \mathbb{R}) \end{equation*}
      🔗
      on each of the intervals \((-\infty,0)\text{,}\) \((0,\infty)\text{.}\)
      🔗
      🔗
    🔗
    🔗
  • We determine \(c\) by plugging in the initial condition
    \begin{equation*} 0 = y(1) = c(1)^{2} - \dfrac{1}{2}, \end{equation*}
    🔗
    so \(c = \dfrac{1}{2}\text{.}\)
    🔗
    Therefore, the solution of the IVP is
    \begin{equation*} \boxed{y(t) =\dfrac{1}{2}t^2-\dfrac{1}{2},\ 0<t<\infty.} \end{equation*}
    🔗
    🔗
🔗
We can also solve this problem by using the definite integral approach (see (2.1.2) ):
\begin{equation*} \mu(t)=\exp{\left(\int_{1}^t \left[-\dfrac{2}{x}\right]\,dx\right)}=\exp\left(-2\ln|t|\right)=t^{-2},\quad 0<t<\infty. \end{equation*}
🔗
and the desired solution on \((0,\infty)\) is obtained from
\begin{align*} y\amp =\dfrac{1}{t^{-2}}\left[0+\int_{1}^{t} x^{-2}\dfrac{1}{x}\,dx \right] \\ \amp =t^{2}\left[\int_{1}^{t} x^{-3}\,dx \right] \\ \amp =t^{2}\left[\left.-\dfrac{1}{2}x^{-2}\right|_{1}^{t} \right] \\ \amp =t^{2}\left[\dfrac{1}{2}-\dfrac{1}{2}t^{-2}\right], \end{align*}
so
\begin{equation*} \boxed{y(x) = \dfrac{1}{2}t^2-\dfrac{1}{2}, \ 0 < t < \infty.} \end{equation*}
