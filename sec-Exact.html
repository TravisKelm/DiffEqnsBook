    Section 2.6 Exact Differential Equations

    Introduction.

    Consider a first-order differential equation
    \begin{equation} M(x,y)+N(x,y)\dfrac{dy}{dx}=0\tag{2.6.1} \end{equation}
    sometimes written in a more symmetric differential form
    \begin{equation} M(x,y)dx+N(x,y)dy=0.\tag{2.6.2} \end{equation}
    🔗
    Observe that any first-order differential equation
    \begin{equation*} y'=f(x,y) \end{equation*}
    can be written as (2.6.1) or (2.6.2) with \(M(x,y)=f(x,y)\) and \(N(x,y)=-1\text{.}\)
    🔗
    However, if it happens that there is a function \(F(x,y)\) such that
    \begin{equation*} F_{x}(x,y)=M(x,y) \text{ and }F_{y}(x,y)=N(x,y), \end{equation*}
    then differential equation (2.6.1) turns into
    \begin{equation*} F_{x}(x,y)+F_{y}(x,y)\dfrac{dy}{dx}=0. \end{equation*}
    🔗
    Such a differential is called exact since its left-hand side is exactly the derivative of \(F(x,y)\) relative to \(x\text{,}\) assuming \(y\) to be a function of \(x\text{,}\) \(y=y(x)\text{,}\) and applying the chain rule.
    🔗
    Thus, in this case, we have:
    \begin{equation*} \dfrac{d}{dx}F\left(x,y(x)\right)=0. \end{equation*}
    🔗
    Consequently, the equation
    \begin{equation*} F(x,y)=c\quad (c\in \mathbb{R}) \end{equation*}
    implicitly defines the general solution of the original differential equation.
    🔗

    Subsection 2.6.1 How Do We Know If a Differential Equation Is Exact?

    In (2.6.1) (or, equivalently, (2.6.2) ), if \(M(x, y)\) and \(N(x, y)\) are defined and have continuous first partial derivatives in an open region in the \(xy\)-plane, then
    \begin{equation*} M_{y}(x, y) = F_{xy}(x, y) \text{ and }N_{x}(x, y) = F_{yx}(x, y). \end{equation*}
    🔗
    By the assumption of the continuity of the first partial derivatives of \(M\) and \(N\text{,}\) the second derivatives of \(F\) are also continuous. Then, by equality of mixed partial derivatives
    \begin{equation*} F_{xy}(x, y) = F_{yx}(x, y) \end{equation*}
    (see, e.g., [cross-reference to target(s) "Stewart" missing or not unique] ), we obtain the following condition that must be satisfied for the differential equation to be exact:
    \begin{equation*} M_{y}(x, y) = N_{x}(x, y). \end{equation*}
    🔗
    🔗

    Subsection 2.6.2 How Do We Solve an Exact Differential Equation?

    Provided condition () holds, the equation is exact and, since we are looking for a function \(F(x,y)\) such that \(F_{x}(x,y)=M(x,y)\text{,}\) we have
    \begin{equation*} F(x,y)=\int M(x,y)\,dx+g(y). \end{equation*}
    🔗
    Recall that when integrating a function of \(x\) and \(y\) relative to \(x\text{,}\) we obtain instead of the integration constant an arbitrary function of \(y\text{,}\) \(g(y)\text{.}\)
    🔗
    Since, we also have the condition that \(F_{y}(x,y)=N(x,y)\text{,}\) we differentiate the former expression relative to \(y\text{,}\) obtaining a differential equation for \(g(y)\text{,}\) which we solve for \(g(y)\) and, thus, obtain \(F(x,y)\text{.}\)
    🔗
    Observe that the order of finding \(F(x,y)\) can be different: we can, first, use the condition \(F_{y}(x,y)=N(x,y)\text{,}\) obtaining
    \begin{equation*} F(x,y)=\int N(x,y)\,dy+h(x), \end{equation*}
    where, since we are integrating with respect to \(y\) we obtain an arbitrary function of \(x\text{,}\) \(h(x)\text{,}\) instead of the integration constant. Since \(F_{x}(x,y)=M(x,y)\text{,}\) we differentiate the former expression with respect to \(x\text{,}\) obtaining a differential equation for \(h(x)\text{,}\) which we solve for \(h(x)\) and, thus, obtain \(F(x,y)\text{.}\)
    🔗

    Example 2.6.1.

    Find a general solution of
    \begin{equation*} y+xy'=0. \end{equation*}
    🔗
    Solution.
    As is easily checked this DE is exact (also separable and linear!). Since \(M(x, y) = y, \ N(x, y) = x\text{,}\) and
    \begin{equation*} M_{y}(x, y) = 1 = N_{x}(x, y), \end{equation*}
    the equation is exact.
    🔗
    Now we are looking for a function \(F(x,y)\) such that \(F_{x}(x,y)= M(x, y) = y\text{,}\)
    \begin{equation*} F(x,y)=\int y\,dx=xy+g(y), \end{equation*}
    where \(g(y)\) is a function to be determined.
    🔗
    Since \(F_{y}(x,y)=N(x, y) = x\text{,}\) we have that
    \begin{equation*} x = F_{y}(x, y) = \frac{\partial}{\partial y}(xy + g(y)), \end{equation*}
    or
    \begin{equation*} x+g'(y)=x. \end{equation*}
    🔗
    This gives, \(g'(y)=0\text{,}\) and thus, \(g(y)=0 + c_{1}\) ( \(c_{1}\in\mathbb{R}\) ).
    🔗
    Therefore, \(F(x,y)=xy+c_{1}\) and the general solution is given by \(F(x, y) = c\text{,}\) so we have
    \begin{equation*} xy + c_{1} = c, \end{equation*}
    or
    \begin{equation*} \boxed{xy=c\quad (c\in\mathbb{R}).} \end{equation*}
    🔗
    🔗
    🔗

    Example 2.6.2.

    Find a general solution of
    \begin{equation*} (2x^{2}-y^{2})dx + (2y^{2}-2xy)dy = 0. \end{equation*}
    🔗
    Solution.
    Since \(M(x, y) = 2x^{2}-y^{2}, \ N(x, y) = 2y^{2}-2xy\) and
    \begin{equation*} M_{y}(x, y) = -2y= N_{x}(x, y), \end{equation*}
    the equation is exact.
    🔗
    Next, find \(F(x, y)\text{.}\)
    \begin{align*} F(x,y)&= \int M(x,y)dx \\ &= \int(2x^{2}-y^{2})dx \\ &= \frac{2}{3}x^{2}-xy^{2}+g(y) \text{.} \end{align*}
    🔗
    Now, since \(N(x, y) = F_{y}(x, y)\text{,}\) find the partial derivative of \(F(x, y)\) with respect to \(y\) and set it equal to \(N(x, y)\) to find \(g(y)\text{.}\)
    🔗
    Since \(\displaystyle N(x, y) = F_{y}(x, y)\text{,}\) we have
    \begin{align*} 2y^{2} - 2xy&= -2xy + g'(y). \end{align*}
    We then solve for \(g(y)\) .
    \begin{align*} g'(y)&= 2y^{2} \\ g(y)&= \frac{2}{3}y^{3} (+c_{1}) \text{.} \end{align*}
    🔗
    So, \(F(x, y) = \frac{2}{3}x^{3} - xy^{2} + \frac{2}{3}y^{3}\text{.}\)
    🔗
    We do not need to include the \(+c_{1}\) in the above calculation, because our solution is of the form \(F(x, y) = c\text{,}\) and so the constants will be combined in the solution. The general solution is given by the equation
    \begin{equation*} \boxed{\frac{2}{3}x^3 - xy^2 + \frac{2}{3}y^3 = c\quad (c\in\mathbb{R}).} \end{equation*}
    🔗
    🔗
    🔗

    Example 2.6.3.

    Solve the initial value problem
    \begin{equation} y^{3} + 3xy^{2} \frac{dy}{dx}= 0,\quad y(1) = -2.\tag{2.6.3} \end{equation}
    🔗
    Solution.
    Since \(M(x, y) = y^{3}, \ N(x, y) = 3xy^{2}\) and
    \begin{equation*} M_{y}(x, y) = 3y^{2}=N_{x}(x, y), \end{equation*}
    the equation is exact.
    🔗
    Next, find \(F(x, y)\text{.}\)
    \begin{align*} F(x,y)&= \int N(x,y)dy \\ &= \int 3xy^{2} dx \\ &= xy^{3} + h(x) \text{.} \end{align*}
    🔗
    Now, since \(M(x, y) = F_{x}\text{,}\) find the partial derivative of \(F(x, y)\) with respect to \(x\) and set it equal to \(M(x, y)\) to find \(h(x)\text{.}\) We have
    \begin{align*} y^{3}&= y^{3} + h'(x) \\ h'(x)&= 0 \\ h(x)&= 0+c_{1} \text{.} \end{align*}
    🔗
    As noted at the end of Example 2.6.2, we can omit the \(+c_{1}\) in the calculation, so \(F(x, y) = xy^{3}\text{,}\) and the general solution is given by \(F(x, y) = c\text{,}\) so we get the equation
    \begin{equation*} xy^{3} = c. \end{equation*}
    🔗
    Lastly we can find the value of \(c\text{:}\) Use \(y(1) = -2\text{,}\) which gives \(x = 1, y = -2\text{.}\) Substituting into the general solution gives
    \begin{align*} 1(-2)^{3}&= c \\ c&= -8 \text{.} \end{align*}
    🔗
    So, in conclusion, the solution to equation (2.6.3) is
    \begin{equation*} \boxed{xy^3 = -8.} \end{equation*}
    🔗
    Note: The astute reader might have noticed that this equation is also separable. You should verify that solving as a separable equation gives the same results.
