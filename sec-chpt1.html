    Section 1.1 Chapter 1

    Definition 1.1.1.

    An equation relating an unknown function and one or more of its derivatives is called a differential equation (or DE for short).
    🔗
    A DE in one independent variable is called an ordinary differential equation (short, ODE). A DE in several independent variables is called a partial differential equation (or PDE).
    🔗
    The order of a differential equation is the order of the highest derivative of the unknown function in the equation.
    🔗
    🔗

    Example 1.1.2. Examples of Differential Equations.

    1. \(\dfrac{dy}{dt}=y\) is a first-order ODE in independent variable \(t\text{.}\)
      🔗
    2. \(\dfrac{\partial u}{\partial x}+\dfrac{\partial u}{\partial y}=0\) is a first-order PDE in two independent variables, \(x\) and \(y\text{.}\)
      🔗
    3. \(y^{(3)}+xy'+y^2=e^{-x}\) is a third-order ODE in independent variable \(x\text{.}\)
      🔗
    4. \(u_{t}=u_{xx}+u_{yy}\) is a second-order PDE in three independent variables \(t\text{,}\) \(x\) and \(y\) (this is a two-dimensional heat equation).
      🔗
    🔗
    🔗

    Note 1.1.3. Notations for Derivatives.

    The prior examples demonstrate various notations for derivatives.
    🔗
    🔗
    Recall the following. Let \(y\) be a function of \(x\text{,}\) so \(>y = y(x)\text{.}\)
    1. For first derivatives, the notations \(\dfrac{dy}{dx}\) and \(y'\) are equivalent.
      🔗
    2. For higher order derivatives, the notations \(\dfrac{d^n y}{dx^n}\) and \(y^{(n)}\) are equivalent.
      🔗
    🔗
    Let \(u\) be a function of \(x\) and \(y\text{.}\) So \(u = u(x, y)\text{.}\) Then we may represent the partial derivatives by, for example:
    \begin{equation*} \dfrac{\partial u}{\partial x} \text{ and } \dfrac{\partial^2 u}{\partial^2 y} \end{equation*}
    or, equivalently
    \begin{equation*} u_x \text{ and } u_{yy}. \end{equation*}
    🔗

    Warning 1.1.4. This book only deals with ODEs.

    For the remainder of this textbook, we will only consider ordinary differential equations.
    🔗
    🔗
    The most general form of an \(n^{\text{th}}\)-order ordinary differential equation with independent variable \(x\) and unknown function (or dependent variable) \(y=y(x)\) is
    \begin{equation} F\bigl(x,y,y',y'',\dots,y^{(n)}\bigr)=0,\tag{1.1.1} \end{equation}
    where \(F\) is a function of the \(n+2\) variables.
    🔗
    An \(n^{\text{th}}\)-order differential equation solved relative to the highest derivative (which may not be possible) is said to be in standard (or normal) form:
    \begin{equation} y^{(n)}=f\bigl(x,y,y',y'',\dots,y^{(n-1)}\bigr),\tag{1.1.2} \end{equation}
    where \(f\) is a function of the \(n+1\) variables \(x, y, y', \dots, y^{(n)}\text{.}\)
    🔗
    Thus, in Example 1.1.2 the prior examples, Item 1 is in standard form and Item 3 is not.
    🔗

    Definition 1.1.5. Solution of a Differential Equation.

    A function \(y=y(x)\) is called a solution of differential equation (1.1.1) on an interval \(I\) if it is \(n\) times differentiable on \(I\) and
    \begin{equation*} F\bigl(x,y(x),y'(x),y''(x),...,y^{(n)}(x)\bigr)=0 \end{equation*}
    for all \(x\in I\text{.}\)
    🔗
    The function \(y=y(x)\) is also said to satisfy differential equation (1.1.1) on the interval \(I\text{.}\)
    🔗
    🔗

    Example 1.1.6.

    For any \(c\in \R\text{,}\) the function \(y=ce^t\) is a solution of the DE
    \begin{equation*} \dfrac{dy}{dt}=y \end{equation*}
    on the interval \((-\infty,\infty)\)
